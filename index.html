Welcome to INSAcity !

INSAcity is a very basic city simulation game, developped in Pascal, and using SDL for graphics.

The source code has been published here mainly to help students that have to do projects using Pascal, which isn't always the simplest thing to do, and to show that even with a language depicted as "obsolete" by many, people can make performant, functional and fun programs :). We hope you enjoy it !

Some INSAcity's features:

*A resource system including money (taxes !), food, shelter, work & environment. *Buildings that has an impact on those resources *An RPG-like system with XP, LVLs, QUESTS; giving the mayor XP (and eventually a LVL UP) as he does great things like getting 42 citizens in his city. *A basic save system (the mayor's LVL and XP are saved in a file and loaded when starting a game with the same mayor's name) *Pollution/Population's density maps.
